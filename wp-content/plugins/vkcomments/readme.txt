Displays vk.com comments widget
== Description ==

Displays vk.com comments widget with or instead of standart wordpress comments

== Installation ==

1. Upload  folder `vkcomments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
